# Designing Innovative Products for Empowering the Delivery Workforce

## Project Overview
This project focuses on developing an innovative smart product that supports delivery personnel by helping them manage the weight of packages during transportation. The system uses a load cell sensor to measure carried load in real time and provides continuous monitoring to prevent excessive weight burden.

The aim of this project is to improve the working conditions of delivery personnel by reducing physical strain, improving safety, and supporting efficient package handling in modern logistics systems.

---

## Problem Statement
Delivery workers in logistics and e-commerce sectors often carry multiple packages during daily operations. Continuous carrying of heavy loads can lead to fatigue, reduced efficiency, and long-term physical stress.

There is a need for a smart assistive system that can:
- Monitor carried package weight
- Prevent overload conditions
- Improve worker safety and comfort
- Support efficient delivery operations

---

## Working Principle
1. Packages are placed inside the delivery bag or carrying system.
2. The load cell continuously measures the applied load.
3. The HX711 amplifier converts the sensor signal into digital data.
4. The microcontroller processes the measured weight.
5. The current load is displayed to the user.
6. If the load crosses a predefined safe limit, an alert is triggered.
